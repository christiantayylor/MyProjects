### How to run the tool:
1. Clone the Repository
```
git clone https://gitlab.com/wgu-gitlab-environment/student-repos/ctay669/d424-software-engineering-capstone/-/tree/main-UpdatedProject?ref_type=heads 
cd VacationProject
```
2. Create the environment file
```
cd server
```
Create `server/.env` with this content:
```
# GeoDB Cities API (existing)
RAPIDAPI_KEY=e16ffef899mshacd2b64325d9b60p1e752fjsn2a44e318e697
RAPIDAPI_HOST=wft-geo-db.p.rapidapi.com

# Booking.com API (hotel search via RapidAPI)
BOOKING_API_KEY=e16ffef899mshacd2b64325d9b60p1e752fjsn2a44e318e697
BOOKING_API_HOST=booking-com.p.rapidapi.com

# Flight Fare Search API (RapidAPI)
FLIGHT_API_KEY=e16ffef899mshacd2b64325d9b60p1e752fjsn2a44e318e697
FLIGHT_API_HOST=flight-fare-search.p.rapidapi.com

# Booking.com Excursion API
EXCURSION_API_KEY=e16ffef899mshacd2b64325d9b60p1e752fjsn2a44e318e697
EXCURSION_API_HOST=booking-com15.p.rapidapi.com

# Currency for bookings
DEFAULT_CURRENCY=USD
```
3. Initialize the database<br>
While still inside the `server` folder, run:
``` 
node src/scripts/initDb.js
```
You should see `All database tables initialized!`<br>
4. Go back to root folder and install all dependencies
``` 
cd ..
npm run install:all
```
This installs packages for the root, `client`, and `server` all at once. It can take a few minutes.<br>
5. Start the app
``` 
npm start
```
This launches both the backend and the frontend together using `concurrently`. 
You will see two streams of output in the same terminal - yellow `[SERVER]` logs and cyan `[CLIENT]` logs. 
After a few seconds the React app will automatically open at `http://localhost:3000`.
### A few things to note for first-time setup:
- The database file (`server/database/vacation.db`) is created automatically by Step 3.<br>
- You only need to run `initDb.js` once. Every subsequent time you just run `npm start` from the root folder.<br>
- If the browser doesn't open automatically, navigate to `http://localhost:3000` manually.
- To stop the app, press `Ctrl + C` in the terminal.
- Every time after the first setup, starting the app is just one command from the `VacationProject` root folder: `npm start`.
